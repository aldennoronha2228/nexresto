# NexResto Menu UI

A high-end, print-inspired editorial design system for a restaurant menu interface.

## Highlights
- **Creative North Star:** "The Silent Maître D'" - authoritative yet hushed, guiding the guest through a culinary journey without the noise of traditional UI.
- **Colors:** Deep obsidian and natural earth tones (Soft Sage & Muted Clay), mimicking the interior of a dimly lit, boutique establishment.
- **Typography:** Extreme typographic scale and intentional asymmetry combining Noto Serif and Inter.
- **Styling Details:** 
  - Tonal depth over structural lines
  - Sharp corners (0px border radius) to convey knife-edge precision
  - Custom provenance tags

## Repository Contents
- `code.html`: The main UI layout of the menu using Tailwind CSS.
- `DESIGN.md`: Detailed design specifications and rules.
- `screen.png`: Screenshot of the finished UI.

## Technologies Used
- HTML5
- Tailwind CSS
- Google Fonts (Noto Serif, Inter)
